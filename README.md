Команды для запуска.
```npm install``` — Установка необходимых зависимостей.
```npm run start ```— запуск проекта с отдачей статики локальным сервером (localhost:3000).
```npm run dev``` — запуск проекта для разработки сборщиком parcel (localhost:1234).
